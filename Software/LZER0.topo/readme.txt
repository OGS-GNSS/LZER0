The SBC running the LZER0 node for monitoring needs:
- a Linux user called lzer0
- a bin dir included into the lzer0 home with the content of the LZER0.mono/bin dir. The dir path must be included in the system path.
- a version of RTKLIB installed
- a crontab that runs all the script (see the crontab here availabkle)
- Warning: this version works needs also the code included in the microcontroller_scripts dir
